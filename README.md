# Bridge-between-operational-technology-OT-and-information-technology-IT-environments.
End-to-end IIoT architecture for Industry 4.0, integrating analogue sensors (FID/TCD) with Cloud Analytics. Focus on IT/OT convergence, cybersecurity (X.509/ISO 27001), Edge Computing, and data governance (LGPD/GDPR) to optimise operational efficiency.
